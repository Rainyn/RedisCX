# RedisCX

![logo](https://i.loli.net/2018/07/10/5b447752b3020.png)

### Feature

**What's important!**

* Support multiple connections.
* Optimize searching for keys in Big-Data dbs.
* Support reserving search history.
* More lighter UI and simpler operation.
